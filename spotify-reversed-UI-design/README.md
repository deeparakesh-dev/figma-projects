# Spotify Reversed UI — Figma Redesign

A custom visual reimagining of Spotify’s interface, designed entirely from scratch in Figma.  
This project explores a **dark-neon-green-spotify-cyber-inspired-theme** while maintaining clarity, hierarchy, and consistency across screens.

## Pages Designed
- Login Page  
- Dashboard
- Search Page  

## Design Focus
- Dark UI hierarchy and contrast
- Consistent visual language across multiple pages
- Controlled use of accent color
- Iterative refinement of layout and components
- Balance between aesthetics and usability

## Components
- Reusable cards and list items  
- Navigation sidebar and top bar  
- Search inputs and interactive states  
- Media player UI  

## Tools
- Figma
- Iconify
- Chrome(extraction of images)

## Files
- Figma site file (`spotify-redesigned-ui.site`)


> This project is a UI exploration and redesign concept, created for learning and practice purposes only.
